# 大模型测试用例大全

> 整理自大量网络评测文章、社区讨论和实测报告（截至 2026-06）
> 这些题目的共同特点是：人类一听就懂，但 AI 经常一本正经地答错。

---

## 一、数字与数学陷阱

| # | 题目 | 翻车点 / 说明 |
|---|------|--------------|
| 1 | **9.11 和 9.9 哪个大？** | 模型受 token 化影响，把 9.11 当版本号处理，认为 11 > 9 |
| 2 | **9.11 和 9.8 哪个大？** | 变体题，同样翻车，认为 11 > 8 |
| 3 | **13.8 和 13.11 哪个大？** | 变体题，源自综艺节目分数争议，模型同样翻车 |
| 4 | **−9.11 和 −9.9 哪个大？** | 负数排序方向搞反 |
| 5 | **"strawberry" 里有几个字母 r？** | Tokenizer 切为 straw + berry，数错（所有 r 类问题的鼻祖） |
| 6 | **"strawberry" 里有几个字母（所有字母）？** | 变体，模型常答 9 或 8，实际是 10 |
| 7 | **"mississippi" 里有几个 s？几个 i？几个 p？** | 变体数数题，连续重复字母更容易数错 |
| 8 | **"banana" 里有几个 a？** | 简单变体，很多模型仍出错 |
| 9 | **"llama" 里有几个 l？** | 数首尾重复字母，容易数漏 |
| 10 | **用 5、5、5、12 算 24 点** | 模型会给出 2.6×5=24、25-12=24 等一本正经的胡说八道 |
| 11 | **我有 6 个鸡蛋，煮了 2 个、煎了 2 个、吃了 2 个，还剩几个？** | 没意识到"煮的/煎的/吃的"可能是同一批蛋 |
| 12 | **你在跑步比赛中超过了第二名，你现在第几名？** | 很多模型答第一名（正确答案：第二名） |
| 13 | **你在跑步比赛中超过了最后一名，你现在第几名？** | 变体陷阱——比赛中不可能超过最后一名（最后一名之后没人） |
| 14 | **3 个人 3 天喝 3 桶水，9 个人 9 天能喝几桶？** | 经典比例陷阱，很多人/模型直接答 9 桶（正确答案：27 桶） |
| 15 | **蝙蝠和棒球一共 1.10 美元，蝙蝠比棒球贵 1 美元，棒球多少钱？** | 脱口而出 0.10 美元（正确答案：0.05 美元）。这是认知偏误经典题 |
| 16 | **一年中有多少个月有 28 天？** | 只想到 2 月，答 1 个月。正确答案：12 个月（所有月份都有 28 号） |
| 17 | **3 + 5 × 2 = ？** | 四则运算优先级，模型先算加法得 16（正确顺序：先乘后加，得 13） |
| 18 | **6 × 13 = ？** | 研究显示模型在思维链中会算错简单的乘法，如给出 68（正确答案 78） |
| 19 | **我有 10 个苹果，给了朋友 3 个，又买了 5 个，又被偷了 2 个，还剩几个？** | 多步加减法，模型容易跳步算错 |

## 二、物理常识与体感逻辑

| # | 题目 | 翻车点 |
|---|------|--------|
| 20 | **洗车店离我家只有 50 米，该走路去还是开车去？** | 模型分析"50 米走路更省油"——但你是去洗车的，车不开过去怎么洗？ |
| 21 | **左右手各握一支笔的两端，松开右手会怎样？** | 说"笔从右手中掉落"——忘了左手还握着另一端 |
| 22 | **月球上同时放下羽毛和保龄球，哪个先落地？** | 套地球直觉（空气阻力），忘了月球是真空 |
| 23 | **一个没底的杯子，杯口还封住了，怎么喝水？** | 模型说"结构不具备装水条件"——倒过来用，没底那面朝下就是底 |
| 24 | **3 米宽、4 米高的城门，5.5 米的竹竿能通过吗？**（竹竿进城问题） | 考验空间推理，很多模型直接说不能，实际斜着可以 |
| 25 | **握住一支笔，转 180° 后笔尖指向哪儿？** | 空间想象题，模型容易搞反方向 |
| 26 | **一公斤棉花和一公斤铁，哪个重？** | 经典物理常识陷阱，模型有时被"铁比棉花重"的常识误导 |
| 27 | **井深 10 米，蜗牛白天爬 3 米，晚上滑 2 米，几天爬出？** | 很多模型答 10 天（实际：第 8 天就到顶了，因为最后一天不会下滑） |
| 28 | **一个房间里有 10 支蜡烛，吹灭了 3 支，还剩几支？** | 模型答 7 支——但蜡烛还在那里，数量不变，10 支 |
| 29 | **一块蛋糕切 3 刀，最多能切成几块？** | 平面切答 6 块或 7 块（正确答案：空间切 8 块） |

## 四、逻辑推理与规划

| # | 题目 | 说明 |
|---|------|------|
| 52 | **狼、羊、白菜过河（农夫每次只能带一样，狼吃羊、羊吃白菜）** | 经典约束满足问题，最少 7 次。考核多步规划能力 |
| 53 | **过河问题变体：两只鸡一次可运两只，最优几次？** | 网友定义的"劣效比率"（crapness ratio）——明明可以一次运完，模型却给出复杂方案 |
| 54 | **从东岸运到东岸（同岸），需要几次？** | 测试模型是否有"常识"意识到不需要过河，但模型照算不误 |
| 55 | **农夫带一只羊站在河边，船可容纳一人一羊，最少的船把人和羊送到对岸？** | 菲尔兹奖得主 Gowers 测试题——只需要 1 次，但 Claude 等模型答 3 次 |
| 56 | **骑士（只说真话）、无赖（只说假话）、间谍（可真可假）三人推理** | 经典逻辑谜题：「A 说：我是骑士。B 说：A 说的是真话。C 说：B 是间谍。」 |
| 57 | **鸡兔同笼（35 头、94 脚，鸡兔各几只？）** | 基础数学推理，要求逐步计算 |
| 58 | **三个盒子标签都贴错了（苹果/橘子/混装），从一个盒子拿一个水果，判断所有盒子内容** | 经典标签推理 |
| 59 | **理发师悖论（罗素悖论）：理发师只为不自己刮胡子的人刮胡子，那么他自己刮不刮？** | 自指逻辑 |
| 60 | **意外绞刑悖论** | 自指 + 预期定义 |
| 61 | **四个人过桥，只有一把手电筒，每次最多两人，过桥速度不同（1、2、5、10 分钟），最少几分钟全过？** | 经典过桥问题，最短 17 分钟。模型常给出错误的贪心策略答案 |
| 62 | **1000 瓶酒中有一瓶毒药，用最少的小白鼠在 24 小时内找出毒酒** | 二进制编码思维（需要 10 只），模型常缺乏二进制搜索意识 |
| 63 | **假设 3 条狗和 3 个人要过河，船每次最多载 2 个，人不在时狗不能多于人（否则狗会攻击），求方案** | 变体过河，更复杂的约束条件 |
| 64 | **如果你在比赛中超过了第二名，你现在第几名？**（与 12 题重复，已归入第一章） | — |

## 五、语言理解与语义歧义

| # | 题目 | 翻车点 / 说明 |
|---|------|--------------|
| 65 | **"客户端起酒杯讲了祝酒词"，是谁讲了祝酒词？** | 分词器把"客户端"切成一个词，模型答"客户端（软件）讲祝酒词" |
| 66 | **女孩小美有 2 个弟弟和 3 个姐妹，小美的弟弟有几个姐妹？** | 正确答案：4 个（小美本人也是弟弟的姐妹），模型常答 3 个 |
| 67 | **农夫有 10 头牛，sells all but 3，还剩几头？** | 不理解 idiomatic 的 "all but 3 = 剩 3 头"，按字面做减法 |
| 68 | **两个父子钓了三条鱼，每人分到一条，怎么回事？** | 答案是爷爷、爸爸、儿子（三个人，两种"父子"关系） |
| 69 | **写 5 个描述某事物的词，每个词不含字母 e** | Token 级别的模型"看不见"字符粒度，几乎做不到 |
| 70 | **小明说"我昨天没去图书馆"，监控显示他去了；又说"我说谎时一定在说真话"——判断真伪** | 元语言悖论 + 事实判断，模型需区分"描述性语句"和"规则性语句" |
| 71 | **"小明的妈妈有 5 个孩子，分别叫 April、May、June、July，第五个叫什么？"** | 答案：小明。模型常被英文月份名绕晕 |
| 72 | **"从 1 到 100，数字 9 出现了多少次？"** | 模型常只算个位上的 9（10 次），忘了十位上也有 9（共 20 次） |
| 73 | **"一辆火车从 A 站开往 B 站，全程 300 公里，时速 100 公里/小时，同时一只鸟以 150 公里/小时在两站间往返飞，火车到站时鸟飞了多远？** | 经典"两列火车与鸟"问题，很多人绕进去算无穷级数，实际上直接用时间乘速度 1.5h×150=225km |

## 六、中文特色测试

| # | 题目 | 说明 |
|---|------|------|
| 74 | **汉字笔画数（如"凹"、"凸"、"噩"各几画）** | 中文特有，很多模型数不对（凹 5 画，凸 5 画，噩 16 画） |
| 75 | **汉字字谜**（如"上不在上，下不在下，人有它大，天没它大"打一字 → 答案：一） | 考验中文文化与文字理解 |
| 76 | **谐音梗脑筋急转弯**（布怕什么？纸怕什么？→ 布怕一万，纸怕万一） | 中文谐音理解 |
| 77 | **绕口令理解**（如"四是四，十是十，十四是十四，四十是四十"） | 中文语音/语义理解 |
| 78 | **写一段话，每个字都要包含"口"字** | 对 Token 级别模型是极大挑战（如"口、回、品、田、古"等） |
| 79 | **用"吃"和"饭"造一个句子，句中不能出现"吃饭"这两个连在一起的字** | 考验"不能出现特定连续词"的约束理解 |
| 80 | **"斑马的皮肤是什么颜色？"** | 模型常答"肉色"——实际斑马皮肤是黑白色的 |
| 81 | **"林黛玉倒拔垂杨柳"是《红楼梦》中的情节吗？** | 这是鲁智深的故事，模型可能被诱导编造红楼梦版 |
| 82 | **"秦始皇在公元前 200 年统一六国"——这个说法对吗？** | 事实错误检测（实际是公元前 221 年），看模型能否主动纠正 |
| 83 | **"请写出一个句子，每个字读音都包含 'i'（如 拼音 yi/li/ji）"** | 中文拼音约束，对模型难度极大 |

## 十二、反常识推理

| # | 题目 | 说明 |
|---|------|------|
| 112 | **如果所有猫都会飞，Tom 是猫，Tom 会飞吗？** | 考核模型是否能进行反事实推理（正确答案：会飞——即使现实中猫不会飞） |
| 113 | **如果说"任何不包含自身的集合都是正常的"，那"所有正常集合的集合"是否正常？** | 罗素悖论变体 |
| 114 | **张三比李四高，王五比张三矮，谁最高？** | 简单逻辑排序，但模型有时推理顺序混乱 |
| 115 | **如果"如果下雨我就带伞"成立，现在张三带伞了，是否一定下雨？** | 充分条件和必要条件混淆（带伞还可能因为防晒） |
